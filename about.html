What is PawSafe?

PawSafe is an interactive educational website teaching pet owners, both new and seasoned, how to reduce the spread of zoonotic diseases between animals and people. It features an interactive Q&A, informative articles, prevention toolkit, and learning games that can be used independently or adapted for classrooms, veterinary clinics, and community education.

How it was built

PawSafe is a fully client-side static site built with plain HTML, CSS, and JavaScript - no frameworks, no backend, no tracking. It uses your browser's local storage only to remember your badge progress and dark mode preference.

The content is adapted from an independent study research project on zoonotic diseases, with source material verified against CDC, WHO, AVMA, Mayo Clinic, and Cleveland Clinic publications.
